Position: static (default)
- Элемент располагается в нормальном потоке документа, как обычный блочный или строчный элемент.
- Свойства top, left, right, bottom игнорируются.
- Элемент занимает место в документе согласно его порядку в HTML и размерам.
- Элемент не позиционируется относительно окна или других элементов.

Position: relative Способ позиционирования элемента относительно его нормального положения в документе.
- Элемент сначала занимает место в документе так же, как при static — то есть в обычном потоке.
- Затем к нему применяются смещения с помощью свойств top, right, bottom, left.
- Эти смещения сдвигают элемент относительно его исходного положения, но не влияют на расположение других элементов (то есть место, которое элемент занимал, остаётся занятым).
- Если смещения не заданы (top: 0, left: 0 и т.п.), элемент выглядит как при position: static.

Position: absolute;
Элемент вырывается из нормального потока документа, не занимает места.
- Другие элементы ведут себя так, как будто его нет.
- Положение задаётся свойствами top, left, right, bottom.
- Ориентир (относительно чего считается top, left и т.п.):
- Если есть ближайший родитель с position ≠ static — используется он.
- Если нет — считается относительно body/вьюпорта окна.
Position: fixed;
Элемент вырывается из потока документа (как и при absolute).
Размещается как элемент с position: absolute
но остаётся на месте при прокрутке страницы.
